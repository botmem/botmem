  <files>Makefile</files>
  <action>
Create a Makefile at the repo root with the following targets (all declared `.PHONY`):

- **dev** ‚Äî depends on `up`, then runs `pnpm dev`. This is the single-command "clone to running app" target.
- **up** ‚Äî runs `docker compose up -d --wait` (starts Redis + Qdrant, blocks until health checks pass).
- **ollama-up** ‚Äî runs `docker compose --profile ollama up -d --wait` (starts all services including Ollama).
- **down** ‚Äî runs `docker compose down` (stops services, preserves volumes).
- **status** ‚Äî runs `docker compose ps` (shows service status).
- **clean** ‚Äî runs `docker compose down -v` (removes services AND volumes ‚Äî destructive).

Each target should have a brief comment above it explaining what it does.

Use tabs for recipe indentation (Makefile requirement ‚Äî spaces will not work).
