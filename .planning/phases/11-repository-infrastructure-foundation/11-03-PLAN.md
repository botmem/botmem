<task type="auto">
  <name>Task 2: Configure VPS with Docker, swap, and firewall via SSH</name>
  <files>None (remote VPS configuration)</files>
  <action>
    SSH into the VPS at the IP provided by the user and run the following configuration:

    1. **Configure 2GB swap**:
       ```bash
       ssh root@VPS_IP 'fallocate -l 2G /swapfile && chmod 600 /swapfile && mkswap /swapfile && swapon /swapfile && echo "/swapfile none swap sw 0 0" >> /etc/fstab'
       ```

    2. **Install Docker Engine + Docker Compose plugin**:
       ```bash
       ssh root@VPS_IP 'apt-get update && apt-get install -y ca-certificates curl gnupg && install -m 0755 -d /etc/apt/keyrings && curl -fsSL https://download.docker.com/linux/ubuntu/gpg | gpg --dearmor -o /etc/apt/keyrings/docker.gpg && chmod a+r /etc/apt/keyrings/docker.gpg && echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo $VERSION_CODENAME) stable" | tee /etc/apt/sources.list.d/docker.list > /dev/null && apt-get update && apt-get install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin'
       ```

    3. **Configure UFW firewall**:
       ```bash
       ssh root@VPS_IP 'ufw default deny incoming && ufw default allow outgoing && ufw allow 22/tcp && ufw allow 80/tcp && ufw allow 443/tcp && ufw --force enable'
       ```

    4. **Verify all configuration**:
       ```bash
       ssh root@VPS_IP 'docker --version && docker compose version && swapon --show && ufw status verbose'
       ```

    All commands use direct SSH invocation -- no scripted provisioning tools per user decision.
  </action>
  <verify>
    <automated>ssh root@VPS_IP 'docker --version && docker compose version && swapon --show | grep -c swapfile && ufw status | grep -c "22/tcp"'</automated>
  </verify>
  <done>VPS has Docker Engine, Docker Compose v2 plugin, 2GB swap active, and UFW firewall allowing only ports 22, 80, 443. All verified via SSH.</done>
</task>

    Steps for the user:
    1. Log into Spaceship registrar (https://spaceship.com)
    2. Navigate to DNS settings for botmem.xyz
    3. Create an A record: `@` -> VPS IP address (TTL: 300 seconds)
    4. Create an A record: `www` -> VPS IP address (TTL: 300 seconds)

    6. Verify: `dig +short botmem.xyz` should return the VPS IP
