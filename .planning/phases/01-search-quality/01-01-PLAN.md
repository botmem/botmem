<objective>
Integrate Qwen3-Reranker-0.6B via Ollama generate API to fill the empty 0.30 rerank weight slot in the search scoring formula. After this plan, search results will be reranked using yes/no logprobs from the reranker model, producing meaningfully different result ordering.

Purpose: The scoring formula has a 0.30 weight for rerank that is currently always 0. Filling this slot with actual reranker scores is the highest-impact search quality improvement.
Output: Working reranker integration in OllamaService, updated computeWeights with rerank parameter, updated search flow, unit tests.
</objective>
