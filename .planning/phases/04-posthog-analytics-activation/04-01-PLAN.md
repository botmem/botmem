  <name>Task 2: Add graph_view and graph_node_click frontend events</name>
  <files>apps/web/src/components/memory/MemoryGraph.tsx</files>
  <action>
1. Import `trackEvent` from `../../lib/posthog` at the top of MemoryGraph.tsx.

2. Add a `graph_view` event that fires once when the graph component mounts with data. Use a `useEffect` with a ref guard to fire only once per mount:
   ```typescript
   const hasTrackedView = useRef(false);
   useEffect(() => {
     if (!hasTrackedView.current && data.nodes.length > 0) {
       hasTrackedView.current = true;
       trackEvent('graph_view', {
         node_count: data.nodes.length,
         link_count: data.links.length,
       });
     }
   }, [data.nodes.length, data.links.length]);
   ```
   Place this after the existing useEffect hooks (after the ResizeObserver one).

3. Add a `graph_node_click` event in the `onNodeClick` handler. The existing handler is inline: `onNodeClick={(node: any) => dispatchUI({ type: 'selectNode', node })}`. Update it to also track:
   ```typescript
   onNodeClick={(node: any) => {
     dispatchUI({ type: 'selectNode', node });
     trackEvent('graph_node_click', {
       node_type: node.nodeType || 'unknown',
     });
   }}
   ```
   The `node.nodeType` field is already present on graph nodes (values: 'memory', 'contact', 'group', 'file', 'device').
